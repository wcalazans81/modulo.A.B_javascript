Tipos de Operadores JavaScript

  • Aritméticos
    1. Adição +
    2. Subtração -
    3. multiplicação *
    4. Divisão /
    5. Divisão inteira %
    6. Potênciação **

      7. Ordem de precedência

      1. Em 1º lugar ()
      2. Em 2º lugar potenciação **
      3. Em 3º lugar potenciação multiplicação *, divisão/ e divisão inteira %
      4. Em 4º lugar soma + e subtração
  • Atribuição
    1. sintax de incermento var ++ ou var --
    2. sintax de incermento var // ou var **
    3. sintax de incermento var %% ou var ****
  • Relacionais
    1. 5 > 2 -> true maior
    2. 7 < 4 -> false menor
    3. 8 >= 8 -> true maior ou iqual
    4. 9 <= 7 -> false menor ou iqual
    5. 5 == 5 -> true iqual
    6. 4 != 4 -> false diferênte

      7. Operadores de Identidade

      • 5 == 5 -> true
      • 5 == '5' -> true
      • Para Identidade de tipos 5 === '5' -> false
      • Para Identidade de tipos 5 === 5 -> true
  • Lógicos
    1. Negação !
    2. Conjunção &&
    3. Disjunção ||

      4. Ordem de precedência

      1. Em 1º lugar ! não
      2. Em 2º lugar && E
      3. Em 3º lugar || ou
  • Ternários
  • teste ? "true : false

    • Os Operadores Ternários tem tres operandos
